 # Advanced Monochrome Note App

## Description
A web-based note-taking application with a clean monochrome B/W theme. Manage **Notes**, **Lists**, and **Voice Notes** efficiently with a minimalist, responsive interface.

## Features
- Create, edit, and delete notes and lists
- Voice-to-text notes using browser SpeechRecognition API
- Color-coded notes with 6 grayscale options
- Search notes and lists in real-time
- Floating “Add Note” button for easy note creation
- Editor modal for full editing
- Monochrome Dark/Light theme toggle
- LocalStorage persistence

## Technologies Used
- HTML
- CSS (variables and responsive design)
- JavaScript
- Web Speech API (SpeechRecognition)

## Usage
1. Open `index.html` in a web browser.
2. Click **+** to add a new note or list.
3. Click a note/list to edit or delete.
4. Use the theme toggle button to switch between B/W themes.
5. Use the search bar to filter content.

## Notes
- Audio playback for recordings is a placeholder.
- Works in modern browsers with SpeechRecognition support.
